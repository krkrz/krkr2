●これはなに？

吉里吉里の fps に制限をかけるプラグインです

●使い方

プラグインをロードした時点から吉里吉里メインループの実行頻度が
自動的に制限されます。

 ・トランジション
 ・ContinuousHandler

などの動作もあわせて制限されます。

吉里吉里は、これらの動作を行う場合は、原則 CPU を極力使い切る
設計になっています。このプラグインで制限をかけることで、
CPU 利用率をおさえることができ、余分な消費電力の低減に役立ちます。

以下のプロパティが拡張されます

・System.fpslimit 

　1秒間あたり更新回数の上限を指定します。
　デフォルト値:60

●ライセンス

このプラグインのライセンスは吉里吉里本体に準拠してください。
